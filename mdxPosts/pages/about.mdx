## このサイトについて
このサイトは、Next.js + Tailwind CSS + Chakra UIで作ったTeba_elevenのポートフォリオサイトです。

このページのコンテンツ自体はMDXで記述されているので、Reactのコンポーネントを埋め込むこともできます。

MDXを扱うために、gray-matter + next-mdx-remote + rehype-pretty-code + shikiを使っています。
